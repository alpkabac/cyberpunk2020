Cyberpunk 2020 uses **nine stats**: INT, REF, TECH, COOL, ATTR, LUCK, MA, BODY, EMP.

- Most tasks: **Stat + Skill + 1d10** vs a **Difficulty Value (DV)**.
- **BODY** sets **Body Type (BT)** and **BTM** (wound/damage reduction).
- **EMP** falls when **Humanity** is lost to cyberware; at **EMP 0** the character is not viable as a PC (cyberpsychosis threshold).
