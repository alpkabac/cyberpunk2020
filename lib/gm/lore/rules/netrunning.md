- **Interface** skill drives almost all meaningful Net actions.
- **Deck** stats: CPU, memory (MU), speed, data wall—programs consume **MU** and run in order.
- Runner can **move**, **scan**, **attack ICE**, and **manipulate** remote systems; ICE fights back with its own routines.
- Serious failures: **trace**, **damage**, or **dump shock**—set DV and consequences like any other dangerous task.
