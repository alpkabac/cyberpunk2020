**Single roll:** Stat + Skill + 1d10 vs **DV**

| Task              | DV  |
|-------------------|-----|
| Easy              | 10  |
| Average           | 15  |
| Difficult         | 20  |
| Very Difficult    | 25  |
| Nearly Impossible | 30  |

- **Modifiers:** wounds, gear, environment, time pressure, teamwork (+1–3 each is typical).
- **Opposed:** both roll Stat + Skill + 1d10; high wins; ties favor defender or stalemate (GM).
